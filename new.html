Изготовление железобетонных стоек для опор повышенной долговечности

ООО «Завод ЖБИ» освоил изготовление железобетонных стоек для опор ВЛ 0,4-10 кВ повышенной долговечности по ТУ 5863-007-00113557-94 из тяжелого бетона, модифицированного дисперсией многослойных углеродных нанотрубок.
Модификация нанодисперсной добавкой в виде суспензии многослойных углеродных нанотрубок позволяет получить бетон с характеристиками по трещиностойкости, морозостойкости и водонепроницаемости в два раза выше проектных. Предлагаем к реализации следующие железобетонные стойки: СВ85-2с, СВ85-3с, СВ85-3, СВ95-1, СВ95-2, СВ95-2с, СВ95-3с, СВ95-3, СВ105-3.6, СВ105-5, СВ110-3.5, СВ110-5, СВ90-3.2 предназначенные для опор воздушных линий электропередачи напряжением 0,4-10 кВ повышенной долговечности.
